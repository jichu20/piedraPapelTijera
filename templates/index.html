Nuevo juego

¿Cómo se juega?

    1. Crear un juego nuevo.
    2. Ingrese su nombre.
    3. Empiece a Jugar.
    4. El juego consiste en que el jugador elige una de las tres opciones: piedra, papel o tijera.
    5. La máquina elige una de las tres opciones de forma aleatoria.
    6. Se comparan las elecciones y se determina el ganador.
    7. Si el jugador gana, se le suma un punto. Si la máquina gana, se le suma un punto a ella. Si hay empate, no se suma ningún punto.
    8. El juego termina cuando uno de los dos llega a 3 puntos.
